# Error Catalog

> Every ADO API error encountered and its resolution.
> Check this file FIRST when any ADO call fails. Do not search online.
> Add new errors here after resolving them.

---

## Format

Each entry follows this structure:
- **HTTP Status:** [code]
- **Endpoint:** [which API]
- **Error Message:** [exact or paraphrased]
- **Root Cause:** [what was wrong]
- **Resolution:** [how it was fixed]
- **Date:** [when encountered]
